## 개요
Spring Boot를 활용하여 만든 웹 애플리케이션
- 데이터 등록, 조회, 수정, 삭제 기능을 중심으로 구현
- User 가입 후 전시 목록 조회

## 사용 기술
* Java
* HTML & CSS

## 주요 기능
* 로그인 및 회원가입 <br>
* 전시 목록 조회
  - 배너 표시 -> 클릭 시 전시 사이트로 연결
* 내 정보 조회 및 회원 정보 수정 <br>
